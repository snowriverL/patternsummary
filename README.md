## 设计模式对比

> 工厂模式 
    封装创建的过程，隐藏内部实现
    
> 单例模式
    保证对象在全局环境中的唯一性
    
> 原型模式
    对象的深度copy，保证新对象的创建
    
> 代理模式
    指责的转移，增强
    
> 委派模式
    桥梁传递
    
> 策略模式
    方案多元化
    
> 模板模式
    公共模块，流程的封装处理
    
> 适配器模式
    功能兼容性实现
    
> 装饰器模式
    不改变原有功能的增强处理
